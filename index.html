The Libero Engine [3D]

My most recent project. Coded in C++ - Visual Studio 2020. This is a 3D-rendition of my previously made 2D-Engine.
It provides me with the perfect sandbox to learn about various elements of Game-Engine Development. The main challenge would be that it's programmed in the ECS(Entity-Component-System) system-flow, in which all data is Components and Entities in the world are merely ID's. The Engine uses:
- DirectX11 for rendering
- PhysX for Physics
- ImGui for Editor & User-Interface.
